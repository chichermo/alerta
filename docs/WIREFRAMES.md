# Wireframes (texto)

## Home / Mapa

- Header: logo + botón "Reportar"
- Panel lateral:
  - Filtros por tipo, comuna, nivel
  - Timeline con últimos incidentes
- Mapa central:
  - Marcadores por color según tipo
  - Tooltip con nivel de confianza

## Reportar incidente

- Selector de tipo
- Ubicación (GPS o pin)
- Descripción breve
- Evidencia (foto/video)
- Botón enviar

## Detalle de incidente

- Título + categoría
- Nivel de confianza
- Historial de reportes
- Recomendaciones
