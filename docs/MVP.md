# MVP (2-3 meses) – definido e implementado

## Alcance funcional

- Reporte ciudadano (tipo, ubicación, evidencia)
- Mapa en tiempo real + timeline
- Validación por niveles (observación, alta probabilidad, confirmado)
- Alertas básicas (web)
- Predicción inicial con heurísticas

## Historias clave

1. Como usuario, reporto un incidente en mi comuna.
2. Como usuario, veo incidentes cercanos en un mapa.
3. Como sistema, agrupo reportes similares en un incidente.
4. Como sistema, emito alertas cuando aumenta la confianza.

## Entregables

- API NestJS con endpoints para reportes e incidentes
- Web con mapa, filtros y formulario
- Servicio IA con endpoint de predicción
- Documentación y base legal
