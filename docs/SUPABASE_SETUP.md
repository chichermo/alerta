# Supabase Setup

## 1. Crear proyecto

- Crea un proyecto en Supabase.
- Copia `Project URL` y `anon public` key.

## 2. Ejecutar SQL

Ejecuta el SQL de `supabase/schema.sql` en el SQL Editor.

## 3. Variables de entorno

En Vercel y local agrega:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## 4. Probar

Carga la web y crea un reporte. Debe aparecer en el mapa y timeline.
